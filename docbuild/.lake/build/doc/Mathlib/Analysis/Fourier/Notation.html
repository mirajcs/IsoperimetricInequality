    Type classes for the Fourier transform #

    In this file we define type classes for the Fourier transform and the inverse Fourier transform. We introduce the notation ùìï and ùìï‚Åª in these classes to denote the Fourier transform and the inverse Fourier transform, respectively.

    The notation typeclass for the Fourier transform.

    While the Fourier transform is a linear operator, the notation is for the function E ‚Üí F without any additional properties. This makes it possible to use the notation for functions where integrability is an issue. Moreover, including a scalar multiplication causes problems for inferring the notation type class.
